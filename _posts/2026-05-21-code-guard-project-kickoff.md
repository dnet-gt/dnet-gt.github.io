# CodeGuard

## About the Project
CodeGuard is a project dedicated to helping software companies in Flanders safely, efficiently, and responsibly integrate AI code assistants into their software development processes.

By addressing the challenges associated with Large Language Model (LLM)-based development tools, the project aims to improve developer productivity, enhance code quality, and reduce security risks introduced by AI-assisted software engineering practices.

## Goals
- Raise awareness among software companies about the security risks associated with AI code assistants and LLM-based software development.
- Provide practical knowledge and methodologies for identifying and mitigating these risks.
- Develop and demonstrate secure integration strategies for LLMs, including:
  - Prompt engineering
  - Retrieval-Augmented Generation (RAG)
  - Guardrails based on static and dynamic analysis
- Transfer scientific knowledge to industry partners and contribute to the continuous improvement of secure AI-generated code.
